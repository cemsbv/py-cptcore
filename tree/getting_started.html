Installing py-cptcore

To install pycptcore, we strongly recommend using Python Package Index (PyPI). You can install pycptcore with:

pip install pycptcore

How to import py-cptcore

Getting started with pycptcore is easy done by importing the pycptcore library:

In [1]: import pycptcore

or any equivalent import statement.
