Welcome to py-CPTCore documentation!
=======================================

The `py-CPTCore` library is created by `CEMS BV <https://cemsbv.nl/>`_ and is a public
python wrapper around the CEMS `CPTCore web-API <https://nuclei.cemsbv.io/#/cptcore/api>`_.
